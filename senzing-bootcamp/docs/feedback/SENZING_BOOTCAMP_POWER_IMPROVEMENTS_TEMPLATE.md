## How to Use This File

As you work through the Senzing Boot Camp, document any issues, confusion points, or improvement ideas here. This feedback will be sent to the power author to make the boot camp better for future users.

**When to add feedback**:
- Instructions are unclear or confusing
- Steps don't work as documented
- Missing information or prerequisites
- Workflow feels awkward or inefficient
- Error messages are unclear
- Documentation is outdated
- You discover a better approach
- Tools or templates are missing

**What NOT to include**:
- Issues specific to your data (not general problems)
- Senzing SDK bugs (report those to Senzing directly)
- Feature requests outside boot camp scope
- One-time edge cases
